# Configuration

Configure **avro-nextjs-boilerplate** for different environments.

## Environment variables

| Name                  | Description                              |
| --------------------- | ---------------------------------------- |
| `NEXT_PUBLIC_BASE_PATH` | Base path for GitHub Pages deployment. |

## Build commands

```bash
npm run build
```

```bash
npm run dev
```

> Adjust these commands or add scripts in `package.json` as needed.
