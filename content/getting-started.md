# Getting started

Welcome to **avro-nextjs-boilerplate**.

## 1. Install dependencies

```bash
npm install
```

## 2. Run the dev server

```bash
npm run dev
```

## 3. Deploy to GitHub Pages

- Commit your changes
- Push to `main`
- GitHub Actions will build and deploy automatically

> Tip: You can customize this page by editing `content/getting-started.md`.
